Windows XP 是微软于 **2001 年 10 月 25 日** 正式发布的操作系统。它是 Windows NT 家族中第一个同时面向消费市场和商业市场的版本，终结了此前 Windows 9x 和 Windows NT 双线并行的分裂局面。

XP 的名字来自 "e**X**perience**P**"——体验。微软希望它不只是工具，而是一种感受。

---

## 代号 Whistler：开发史

XP 的开发代号是 **Whistler**，得名于加拿大不列颠哥伦比亚省的惠斯勒滑雪胜地。这个代号贯穿了整个开发周期（1999-2001），期间发布了多个 Beta 版本：

| 版本 | 日期 | 主要变化 |
|---|---|---|
| **Beta 1** (Build 2296) | 2000 年 10 月 | 首次引入 Luna 视觉风格雏形，开始菜单变为双栏 |
| **Beta 2** (Build 2462) | 2001 年 3 月 | Luna 主题接近最终版，ClearType 字体平滑技术加入 |
| **RC1** (Build 2505) | 2001 年 6 月 | 远程桌面、系统还原、防火墙等核心功能就位 |
| **RC2** (Build 2526) | 2001 年 7 月 | 最后测试阶段，主要修 bug |
| **RTM** (Build 2600) | 2001 年 8 月 24 日 | 最终发布制造版，编译号 2600 至今是 XP 的代名词 |

从 Beta 1 到 RTM，UI 经历了巨大变化。早期 Whistler 的界面还残留着 Windows 2000 的影子——直角窗口、灰色按钮、扁平工具栏。Luna 是后来才逐渐浮现的。

XP 实际上有多个版本分支，面向不同市场：

| 版本 | 定位 |
|---|---|
| **Home Edition** | 家庭用户，不支持域、组策略、EFS 加密 |
| **Professional** | 商业用户，完整的网络和安全功能 |
| **Media Center Edition** | 客厅 PC，全屏媒体界面，遥控器操作 |
| **Tablet PC Edition** | 手写笔输入，早期平板电脑 |
| **64-Bit Edition** | Itanium 处理器（非 x64） |
| **Starter Edition** | 发展中国家廉价版，限制同时运行 3 个程序 |
| **Embedded** | ATM、POS 机、医疗设备等嵌入式场景 |

## 内核：NT 5.1

XP 的内核版本是 **NT 5.1**，属于 Windows NT 家族的第五代。从代码谱系来看：

```text
Windows NT 3.1 (1993)
    └─→ NT 3.5 → NT 3.51 → NT 4.0 (1996)
                                └─→ Windows 2000 (NT 5.0, 1999)
                                        └─→ Windows XP (NT 5.1, 2001)
                                                └─→ Windows Server 2003 (NT 5.2)
```

XP 继承了 Windows 2000 的稳定内核——完全 32 位、抢占式多任务、虚拟内存管理、NTFS 文件系统支持——同时大幅改进了兼容性。Windows 2000 被诟病"游戏兼容性差"，XP 用兼容模式（`Compatibility Mode`）解决了这个问题：右键一个程序 → 属性 → 兼容性 → 模拟 Windows 95/98/Me/NT/2000 运行。

**系统要求**（2001 年的硬件标准）：

| 组件 | 最低 | 推荐 |
|---|---|---|
| CPU | Pentium 233 MHz | 300 MHz+ |
| 内存 | 64 MB | 128 MB+ |
| 硬盘 | 1.5 GB 可用空间 | 2 GB+ |
| 显卡 | SVGA 800×600 | SVGA + 支持 DirectX 8 |
| 光驱 | CD-ROM | CD-ROM 或 DVD |

以今天的标准看小得可怜，但在 2001 年，128 MB 内存和 2 GB 硬盘已经是"豪华配置"了。许多旧机器用 64 MB 勉强跑 XP，启动需要三分钟。

## Luna 主题：设计的革命

### 从灰色到蓝色

Windows 2000 及之前的所有版本，窗口都是灰色直角。Luna 彻底打破了这个传统：

- **圆角窗口**：每个窗口的四角都是柔和的圆弧
- **彩色标题栏**：蓝白渐变（默认），橙色高亮按钮
- **红色关闭按钮**：右上角 X 按钮用鲜红色，醒目但不刺眼
- **立体感**：标题栏上半高光（`#0997ff`），下半深蓝（`#003dd7`），配合 1px 浅色边框模拟玻璃反射

这个标题栏渐变的精确色值是本博客 CSS 里最核心的一段代码：

```css
background: linear-gradient(180deg,
  #0997ff 0%,       /* 顶部高光 —— 亮蓝 */
  #0053ee 8%,       /* 快速过渡 */
  #0050ee 40%,      /* 中间色 */
  #0066ff 88%,      /* 渐变到略亮的蓝 */
  #005bff 95%,      /* 接近底部 */
  #003dd7 100%      /* 底部深蓝 —— 收束 */
);
```

8% 处有一个几乎不可见但至关重要的色阶跳变——模拟了玻璃或塑料表面的高光反射带。这就是 Luna 标题栏看起来"有厚度"的秘密。

### 三种颜色方案

| 方案 | 色系 | 说明 |
|---|---|---|
| **默认（蓝）** | `#0050ee` 系 | 出厂设置，最经典的配色 |
| **橄榄绿** | `#7f9a3b` 系 | 绿色调，有些人觉得比蓝色更柔和 |
| **银色** | `#b4b4b4` 系 | 灰色金属质感，接近 Windows 2000 但保留了 Luna 圆角 |

颜色切换的位置：桌面右键 → 属性 → 外观 → 色彩方案。三种方案的标题栏渐变结构相同，只是色相从蓝偏移到绿或灰。

### 开始菜单与任务栏

XP 的开始菜单是一次激进的重设计：

- **双栏布局**：左栏是固定的常用程序列表（IE、Outlook Express 等），右栏是系统位置（我的文档、我的电脑、控制面板等）
- **用户头像**：顶部显示当前用户头像和名称（支持多用户快速切换）
- **蓝色任务栏**：不再是灰色，而是与标题栏同色系的蓝白渐变
- **分组相似任务栏按钮**：同一程序的多个窗口折叠为一个按钮（可选）
- **快速启动栏**：任务栏左侧可以放小程序图标，一键启动
- **系统托盘**：右下角时钟旁边，默认只显示少数图标，其余折叠

双栏开始菜单在发布时争议很大——Windows 2000 用户觉得它"太花哨"，但一旦习惯了，旧版单栏菜单反而显得局促。

## Bliss：世界上观看次数最多的照片

### 拍摄故事

1996 年，前《国家地理》摄影师 **Charles O'Rear** 开车经过加州索诺玛县（Sonoma County）的 12 号公路时，看到了一片起伏的绿色山丘上方飘着蓬松的白云。他把车停在路边，用中画幅胶片相机拍下了这个画面。

这片山丘原本是一个葡萄园，但在拍摄时刚经历了一场虫害，葡萄藤都被拔除了，山坡上只剩下一层翠绿的草皮。O'Rear 后来回忆说："那天早上的光线刚好，云的位置刚好，山丘的绿色也刚好。"

### 如何成为默认壁纸

1999 年，微软在寻找 XP 的默认壁纸。他们通过 Corbis（比尔·盖茨创立的图片库）联系到 O'Rear，买下了这张照片的版权。具体金额从未公开，但传言是六位数美元——如果按观看次数计算，这可能是史上性价比最高的照片授权。

微软的原计划是拍一组纯数字照片，但 Bliss 的中画幅胶片的色彩层次和柔和感让数字照片相形见绌。它被轻微裁切和色彩增强后，放进了 XP 的壁纸文件夹，命名为 `Bliss.bmp`。

### 为什么它如此重要

- **预装在超过 10 亿台设备上**——地球上几乎每一台 2001-2014 年间出厂的 PC 都运行过 XP
- **数以亿计的人每天看到它**——办公室、学校、网吧、家庭电脑
- **它是"数字时代的蒙娜丽莎"**——每个人都见过，但很少有人知道它的故事
- **2014 年 XP 停止支持后**，Bliss 从桌面消失，成为怀旧符号

本博客使用的 `bliss.jpg` 就是这张原始照片。它铺满桌面作为固定背景（`position: fixed; background-size: cover`），让整个博客的视觉锚定在 XP 的语境里。

## 标志性功能

### ClearType

XP 是第一个内置 **ClearType** 子像素渲染技术的 Windows 版本。它利用 LCD 屏幕的 RGB 子像素排列，在字符边缘用彩色条纹欺骗人眼，让字体看起来更平滑、更清晰。

- 在 CRT 显示器上效果一般（CRT 没有固定子像素排列）
- 在 LCD 上效果显著，文字锐利度提升明显
- 中文 ClearType 直到后来才完善（中文字形复杂，子像素渲染容易产生彩色边缘）

### 远程桌面

XP Professional 内置了远程桌面协议（RDP）的服务器端——这是第一次有消费者版本 Windows 可以远程登录。它的前身是 Windows NT 4.0 的 Terminal Server Edition，但在 XP 上变成了开箱即用的功能。只需勾选"允许用户远程连接到此计算机"，另一台电脑就能通过"远程桌面连接"登录。

这个功能在后来改变了整个 IT 行业——远程办公、服务器管理、技术支持几乎都依赖于 RDP 或它的衍生协议。

### 系统还原

XP 引入了系统还原点（System Restore）——自动或手动创建系统状态快照，出了问题可以回滚。它只监控系统文件、注册表和驱动程序，不影响用户文档。

这个功能挽救过无数被恶意软件、驱动冲突或软件安装搞坏系统的电脑。虽然它占用一定的磁盘空间（默认 12%），但在"重装系统"是常态的年代，系统还原是一个巨大的进步。

### Windows 防火墙

XP SP2 引入了 Windows 防火墙——这是微软第一次在消费者版本中内置网络防火墙。SP2 之前，XP 默认没有防火墙，很多计算机在连接到互联网的几分钟内就被 Blaster、Sasser 等蠕虫感染。SP2 默认开启防火墙是 XP 安全史上的转折点。

### 其他值得记住的功能

| 功能 | 说明 |
|---|---|
| **快速用户切换** | 不注销即可切换到另一个用户（Home Edition 特有） |
| **CD 刻录** | 内置光盘刻录功能（不用装 Nero） |
| **自动播放** | 插入 CD/USB 自动弹出操作选项 |
| **主题系统** | `.msstyles` 文件可以更换整个视觉风格 |
| **文件与设置转移向导** | 换电脑时迁移数据（用串口线或网络） |
| **错误报告** | 程序崩溃后弹窗，询问是否向微软发送错误报告 |
| **Windows Update** | 自动下载和安装安全更新 |
| **语言栏** | 多语言输入法切换（对中文/日文/韩文用户至关重要） |

## Service Pack

XP 发布了三个主要服务包：

### SP1（2002 年 9 月）

- 主要修 bug 和安全漏洞
- 添加 USB 2.0 支持
- 首次允许用户隐藏 Internet Explorer、Windows Media Player 等内置程序（响应反垄断判决）
- .NET Framework 1.0 预装

### SP2（2004 年 8 月）

这是 XP 历史上最重要的更新，几乎是一个"新操作系统"级别：

- **Windows 安全中心**：统一管理防火墙、自动更新、杀毒软件状态
- **默认开启防火墙**
- **Internet Explorer 弹出窗口拦截器**
- **无线网络支持大改进**（Wi-Fi Protected Access / WPA）
- **数据执行保护（DEP）**：硬件级别的缓冲区溢出防护
- **Bluetooth 支持**

SP2 让 XP 从"安全噩梦"变成了一个具有基本防护能力的系统。讽刺的是，Blaster 和 Sasser 蠕虫的肆虐恰恰促进了 SP2 的推广——不打补丁就上不了网。

### SP3（2008 年 4 月）

- 将所有 SP2 后的补丁整合为一个安装包
- 引入部分 Windows Vista 的网络功能
- 没有重大新功能——此时微软的精力已转向 Vista 和即将到来的 Windows 7

## 安全与病毒

XP 的流行是一把双刃剑。超过 90% 的市场份额意味着病毒、蠕虫、木马的编写者都把目标对准它：

| 年份 | 事件 |
|---|---|
| 2003 年 8 月 | **Blaster 蠕虫**（又名 Lovsan）：利用 RPC 漏洞，感染数千万台电脑，强制 60 秒倒计时重启 |
| 2004 年 5 月 | **Sasser 蠕虫**：利用 LSASS 漏洞，导致电脑反复重启，全球大量网络瘫痪 |
| 2004 年 | **Mydoom 蠕虫**：史上传播最快的电子邮件蠕虫，高峰期全球 25% 的邮件带毒 |
| 2008 年 | **Conficker 蠕虫**：利用网络服务漏洞，建立了史上最大的僵尸网络之一 |

这些事件反过来推动了 SP2 的强制安装和自动更新的普及。到 2014 年停止支持时，XP 已经积累了超过 10 年的安全更新历史——超过 2000 个安全公告。

## 文化影响

### 默认就等于文化

对整整一代人来说，"电脑"这个词的具体形态就是一台运行 XP 的方盒子显示器，桌面上是 Bliss 绿丘，任务栏是蓝色的，开始按钮是绿色的。

- **网吧时代**：中国 2000 年代中期的网吧几乎清一色 XP + 万象管理软件 + QQ + 传奇/魔兽世界/CS 1.6
- **学校机房**：全国中小学计算机教室的统一配置——联想/方正品牌机 + XP + Office 2003 + 红蜘蛛教学软件
- **办公室标配**：银行柜台、医院挂号、政府窗口——XP 在这些场景里一直服役到 2010 年代末甚至 2020 年代

### 盗版与番茄花园

XP 的盗版文化是中国互联网早期一个不可忽视的侧面。原版 XP 需要激活（Windows Product Activation），催生了大量破解工具和"优化版"定制系统：

- **番茄花园**：最知名的 XP 修改版，集成了驱动、常用软件、主题美化，去掉了激活要求，替换了默认壁纸和 OEM 信息。2008 年作者被逮捕，引发了关于盗版和知识产权的广泛讨论
- **深度技术 / 雨林木风 / 电脑公司特别版**：各路修改版的竞争构成了一个地下生态系统
- **Ghost 还原**：用 Norton Ghost 把整个系统打包成 `.gho` 文件，重装只需 5 分钟，这是中国装机店的标配技能

这些行为在法律上无疑是有问题的，但从技术文化角度看，它们折射出那个时代对"可定制、可掌控"的极致追求——即使是盗版，也是一种对封闭系统的反抗。

### Frutiger Aero 设计风格

XP 的 Luna 主题是 **Frutiger Aero** 设计风格的重要早期代表。这种风格的特征：

- 半透明玻璃质感
- 自然风光（绿色植物、蓝天白云、水流）
- 圆角、光泽、拟物化
- 明亮饱和的色彩

这些特征在 XP 中初现端倪（Luna 标题栏的玻璃高光、Bliss 壁纸的自然主题），在 Windows Vista/7 的 Aero Glass 中达到顶峰，最终在 Windows 8 的扁平化 Metro 设计中被彻底放弃。

2020 年代，Frutiger Aero 作为一种"被怀念的乐观主义美学"重新引起关注——它代表了 2000 年代人们对科技和未来的美好想象。本博客对 XP 的致敬，本质上也是对那个时代的致敬。

## 生命周期之死

2014 年 4 月 8 日，微软正式结束了对 Windows XP 的所有支持。这意味着：

- 没有安全更新
- 没有技术电话支持
- 没有新驱动程序
- Microsoft Security Essentials（免费杀毒）停止更新签名

当时全球仍有约 27% 的桌面电脑运行 XP——超过 5 亿台设备。其中相当一部分是 ATM 机、POS 终端、工业控制系统，它们至今可能还在运行 XP。

XP 的生命周期长达 **12 年 5 个月 14 天**，是微软历史上支持时间最长的操作系统（直到 Windows 10 在 2025 年超越它）。它贯穿了从拨号上网到光纤宽带、从功能手机到智能手机、从 Web 1.0 到 Web 2.0 的整个时代。

## XP 为什么值得怀念

技术上讲，后期的 Windows（比如 Windows 10/11）在安全性、性能、兼容性上都远远超越了 XP。但 XP 有一些后来版本没有的东西：

1. **简洁**：没有广告、没有强制在线账户、没有预装的 Candy Crush、没有"建议"弹窗。你买了电脑，它就是你的。
2. **可控**：控制面板里的一切都看得见摸得着。没有隐藏的注册表黑箱（即使有，至少你知道在哪）。
3. **确定性**：系统不会在你写作时自动重启更新。你选择什么时候更新，它就什么时候更新。
4. **美的诚实**：Luna 的拟物化风格不假装自己是"数字原生"——它承认自己是图形界面，用光泽和圆角让你觉得亲切。相比之下，今天的扁平化设计有时冷得让人不想触碰。

> XP 不是完美的操作系统。但它是很多人第一次"拥有"的操作系统——不是公司配的、不是学校机房的、而是自己攒的电脑上装的第一个正版（或盗版）系统。那种"这是我的电脑"的感觉，后来再也没有了。

## 本博客与 XP

这个博客的所有视觉元素都在致敬 XP：

- **Bliss 壁纸**：Charles O'Rear 的原版照片铺满桌面
- **Luna 标题栏**：`xp.css` 逐像素还原的蓝白渐变
- **窗口凸起边框**：`inset` 多层 box-shadow
- **开始菜单式导航**：横排按钮栏，每个按钮都是 XP 风格
- **树形目录**：TOC 使用 xp.css 的原生 `tree-view` 组件
- **经典图标**：传真封面、信息气泡、XML 文档——这些 PNG 都来自 Windows XP High Resolution Icon Pack

这不是为了怀旧而怀旧。而是因为——在那个点击就是点击、窗口就是窗口、电脑不会偷偷把你文件同步到云端的年代，一切都很确定。这个博客想保留那份确定感。
