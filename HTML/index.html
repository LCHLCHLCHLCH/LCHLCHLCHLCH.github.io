 📻FPGA实现神经网络加速


🗒️ 简介

  硬件层面可能会学习你将处理器和神经网络加速器统一起来，让整个处理器能够对神经网络相关运算更快。软件层面，可能还需要学习神经网络加速算法，用编程语言构造训练模型，然后写入到该处理器上实现图像识别或图像处理等等。

🏇 竞赛&项目

  📣集创芯来riscv杯: 基于开源Riscv处理器蜂鸟E203的内核优化和系统构建
  📣龙芯杯: 从零开始设计CPU ( MIPS指令架构/Longarch指令架构 )
  📣“一生一芯”: 中科院计算所的项目，从零开始手搓CPU ( Riscv指令架构 )

👁️ 先导知识

  📣数字电路: 门电路，组合逻辑/时序逻辑电路
  📣计算机组成原理: 单周期/多周期/流水线/超标量处理器指令执行流程，处理时间相关性和空间相关性的方法
  📣C语言: 了解程序编译执行过程
  📣Verilog语法，或许需要一些python、linux命令行的技能

📖 参考书目

  📣《数字电子技术基础》(阎石)
  📣《手把手教你RISC-V CPU(上)—处理器设计》(芯来科技)
  📣《CPU设计实战》(龙芯中科)

 🚜RISCV处理器设计


🗒️ 简介

  需要学习计算机组成原理相关的东西，了解程序在整个底层的执行的过程，设计优化执行流程的硬件架构，让处理器的运算能力(包括速度)更高，同时也需要了解指令集相关的知识(目前我们这边在研究riscv开源指令集)，以及设计的各个流程该怎么走(模块仿真，指令集测试，程序测试，性能跑分等等)
