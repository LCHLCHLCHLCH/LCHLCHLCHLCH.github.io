# 技术栈

本博客的技术选型：

## 前端

- **XP.css** — Windows XP 风格 CSS 库
- **marked.js** — Markdown 渲染
- **原生 JavaScript** — 零框架，零构建

## 托管

- **GitHub Pages** — 免费静态托管
- 推送即上线，无需构建步骤

## 文件结构

```
├── desktop.html      ← XP 桌面主页面
├── post.html         ← 文章阅读器
├── posts/
│   ├── hello.md
│   ├── about-xp.md
│   └── tech-stack.md
└── icons/            ← XP 高清图标
```

## 为什么不用框架？

1. 更快的加载速度
2. 更少的依赖
3. 更贴近"老式"体验

> Keep it simple.
